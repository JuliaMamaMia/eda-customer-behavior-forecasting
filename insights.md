# Insights (коротко)

## Что сделано
- Предобработка данных: типы, пропуски, дубликаты, выбросы.
- EDA: динамика выручки, распределения, создание признаков.
- Проверка гипотез: сравнение групп клиентов (с выводами и интерпретацией).
- RFM‑сегментация: выделение сегментов и рекомендации по работе с ними.
- Прогнозирование выручки: подготовка временного ряда, выбор модели, валидация, финальный прогноз (ETS).

## Что улучшить (next steps)
- Добавить автоматические data quality checks (например, Great Expectations) и отчёт по качеству данных.
- Упаковать прогноз в функцию/скрипт для регулярного пересчёта.
- Добавить мини‑дашборд (Power BI / Datawrapper) с метриками и сегментами.
